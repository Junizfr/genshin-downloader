## Getting Started

To start this project locally :

1. **Clone the repository**

```bash
git clone git@github.com:Junizfr/genshin-downloader.git
cd genshin-downloader
```

2. Install dependencies

We use Yarn :

```bash
yarn install
```

3. Start the project

To start in development mode :

```bash
yarn start
```
